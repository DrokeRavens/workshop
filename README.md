# Car Repair Workshop 
System created to manage the entry/exit of vehicles, issue nf-e(Brazil), make quotations, generate service reports, and much more. This is the backend with API and business logic.

## Technologies 

* C# / .Net6
* Asp.Net
* RabbitMQ (Messaging)
* SQL Server
* TDD(Test Driven Development)
* GitHub Action (CI)
* Docker
